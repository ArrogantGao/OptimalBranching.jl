Quick Start

This section will provide a brief introduction about how to use the OptimalBranching.jl package.

The maximum independent set problem

We provided two simple interfaces to solve the maximum independent set problem and count the branches used in calculation.

julia> using OptimalBranching, Graphs
julia> g = smallgraph(:tutte){46, 69} undirected simple Int64 graph
julia> mis_size(g) # this gives the size of the maximum independent set19
julia> mis_branch_count(g) # this gives the size of the maximum independent set and the number of branches used in calculation(19, 2)

One can also select different strategies to solve the problem, which inclues

  • AbstractTableSolver to solve the BranchingTable,
  • AbstractSelector to select the branching variable,
  • AbstractMeasure to measure the size of the problem,
  • AbstractSetCoverSolver to solve the set cover problem, and
  • AbstractReducer to reduce the problem.

Here is an example:

julia> using OptimalBranchingCore, OptimalBranching, Graphs
julia> branching_strategy = BranchingStrategy(table_solver = TensorNetworkSolver(), selector = MinBoundarySelector(2), measure=D3Measure(), set_cover_solver = IPSolver())BranchingStrategy
├── table_solver - TensorNetworkSolver(true)
├── set_cover_solver - IPSolver(5, false)
├── selector - MinBoundarySelector(2)
└── measure - D3Measure()
julia> mis_size(g, bs = branching_strategy, reducer = MISReducer())19

One can also use the branch_and_reduce function to solve the problem, which is more flexible.

julia> using OptimalBranchingCore, OptimalBranching, Graphs
julia> branching_strategy = BranchingStrategy(table_solver = TensorNetworkSolver(), selector = MinBoundarySelector(2), measure=D3Measure(), set_cover_solver = IPSolver())BranchingStrategy
├── table_solver - TensorNetworkSolver(true)
├── set_cover_solver - IPSolver(5, false)
├── selector - MinBoundarySelector(2)
└── measure - D3Measure()
julia> branch_and_reduce(MISProblem(g), branching_strategy, MISReducer(), MaxSizeBranchCount)MaxSizeBranchCount(19, 2)
